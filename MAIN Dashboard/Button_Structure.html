1 Door Button Used to enter a facility area/room/section Room 101 Door Details Facility Section This is the room/common area/section button
2 Tool Button Used inside a door to manage that selected area Room Inspection Tool View Selected Door Yellow buttons inside the selected room/common area
3 Task Button Used inside a tool to complete work Check Sink Task Action Selected Tool Task buttons are the work items inside a tool
4 Action Button Used inside a task/tool for extra actions Save Photo or Add Note Action/Popup Selected Task Extra abilities can be defined later
